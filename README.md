# JavaFX Game Collection (Arcade Breakout)

A modular JavaFX arcade-style desktop application featuring multiple playable games in one launcher.

## Features
- Multiple games in a single JavaFX application
- Real-time mechanics (collision detection, movement, enemy spawning, basic AI)
- Custom menus, UI screens, and visual assets
- Sound effects and background music
- Organized object-oriented structure for adding new games

## Tech Stack
- Java
- JavaFX

## How to Run
1. Download or clone the repo
2. Open in Eclipse (or any Java IDE)
3. Ensure JavaFX is configured
4. Run `src/application/Main.java`
